Display any user's Github contributions board on your angular apps with ease

Overview

The component suports info on hover, and click callbacks. Optionally, you can specify an options object to configure the appearence fo the board. The possible customizations size, color scheme, labels size, labels toggle and time span.

Custom size

You can use the cellSize property to set the size in pixels of each board cell. This does not affect the spacing between the cells, therefore the bigger the cells, the denser the board will look. The labels object will allow you to set a specifc size for the day and month labels around the board.

Hiding labels

Through the labels option you can also hide either the day and/or month labels.

Options

The available customization options are:

GithubBoardOptions object

Option Type Default Value Description
cellSize number 10 Defines the size of each individual board cell in pixels.
weeksNumber number 53 Defines the time span shwon on the board in number of weeks.
colorPalette GithubBoardColorPalette Defines the color palette for the board cells.
labels GithubBoardLabelOptions Defines the properties of the labels indicating day and month around the board.

GithubBoardColorPalette object

Option Type Default Value Description
none string #EBEDF0 Defines the color for cells with no contributions.
low string #9BE9A8 Defines the color for cells with 1 - 3 contributions.
medium string #40C463 Defines the color for cells with 4 - 8 contributions.
high string #30A14E Defines the color for cells with 9 - 10 contributions.
higher string #216E39 Defines the color for cells with more than 10 contributions.

The colors must be specified as strings in any HTML suported format (HEX, rgb, rgba...).

GithubBoardLabelOptions object

Option Type Default Value Description
showMonth boolean true Defines whether or not to show month labels.
showDay boolean true Defines whether or not to show day labels.
size number 9 Defines the font size of the labels in pixels.
